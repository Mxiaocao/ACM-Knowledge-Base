# Validation Checklist


每次修改后检查：


## 文件

[ ] 文件存在

[ ] 文件名正确

[ ] 无重复题目文件


## 双链

[ ] 无悬空链接

[ ] 无错误题名链接


## 内容

[ ] 未补懂题没有伪造代码

[ ] 未修改已有题解核心内容

[ ] 每日复盘包含当天题目


## 时间

[ ] 比赛日期正确

[ ] 复盘日期正确

[ ] 入库日期正确

## 仓库真实文件优先检查

[ ] 修改前已扫描当前题目复盘文件
[ ] 修改前已扫描当前知识节点文件
[ ] 修改前已扫描当前每日复盘文件
[ ] 没有因为聊天窗口未提到某题而忽略该题
[ ] 没有因为聊天窗口记忆过期而删除、覆盖、重命名文件
