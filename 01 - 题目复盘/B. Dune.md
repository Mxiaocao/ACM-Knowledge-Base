## 题意简述

给定 $n$ 条向右移动的沙虫和玩家所在的位置 $x$。

第 $i$ 条沙虫初始占据区间：

$$
[l_i,r_i]
$$

向右移动速度为：

$$
v_i
$$

沙虫长度为：

$$
len_i=r_i-l_i
$$

玩家可以选择任意实数时间：

$$
t\ge 0
$$

在时刻 $t$，可能有多条沙虫同时经过位置 $x$。玩家只能召唤这些沙虫中长度最短的一条。

需要选择一个最优时间，使能够召唤的沙虫长度最大，即求：

$$
\max_t
\left(
\min_{\text{时刻 }t\text{ 经过 }x\text{ 的沙虫}}
len_i
\right)
$$

第 $i$ 条沙虫在时间 $t$ 的位置为：

$$
[l_i+v_it,\ r_i+v_it]
$$

经过位置 $x$ 的条件为：

$$
l_i+v_it\le x\le r_i+v_it
$$

因此：

$$
\frac{x-r_i}{v_i}
\le t\le
\frac{x-l_i}{v_i}
$$

考虑 $t\ge 0$，实际经过时间区间为：

$$
\left[
\frac{\max(0,x-r_i)}{v_i},
\frac{x-l_i}{v_i}
\right]
$$

如果：

$$
l_i>x
$$

沙虫一开始就在玩家右侧，并且继续向右移动，永远不会经过 $x$，可以忽略。

经过时间是闭区间。如果一条沙虫在时间 $T$ 结束，另一条沙虫在时间 $T$ 开始，那么恰好在时间 $T$，两条沙虫都经过玩家。

## 数据范围与复杂度判断

关键数据范围：

$$
n\le 10^5
$$

目标时间复杂度应为：

$$
O(n\log n)
$$

不能使用：

- 枚举连续实数时间；
- 每个事件时间重新扫描全部沙虫；

事件排序后，每个事件只处理一次，可以使用扫描线和动态集合完成。

## 核心思路

把每条沙虫经过位置 $x$ 的过程转成一个时间闭区间。

当前活跃沙虫集合只会在两类时间发生变化：

- 某条沙虫开始经过 $x$；
- 某条沙虫结束经过 $x$。

在两个相邻事件时间之间：

- 没有沙虫开始；
- 没有沙虫结束；
- 活跃集合完全不变；
- 当前能够召唤的最短长度也不变。

因此不需要枚举全部连续时间，只需要扫描所有事件时间，并动态维护当前活跃沙虫长度。

使用 `multiset<ll>`：

- 插入开始经过的沙虫长度；
- 删除结束经过的沙虫长度；
- `*cur.begin()` 表示当前所有活跃沙虫的最短长度；
- 对所有可能状态取最大值。

## 从暴力到正解的关键一步

## 正确做法步骤

1. 遍历每条沙虫；
2. 若 $l_i>x$，直接忽略；
3. 计算长度：

   $$
   len_i=r_i-l_i
   $$

4. 计算开始时间：

   $$
   start_i=
   \frac{\max(0,x-r_i)}{v_i}
   $$

5. 计算结束时间：

   $$
   finish_i=
   \frac{x-l_i}{v_i}
   $$

6. 创建两个事件：

   ```text
   (start_i，加入，len_i)
   (finish_i，删除，len_i)
   ```

7. 用分数交叉相乘将事件按时间排序；
8. 使用 `i、j` 找出同一时间的事件段 `[i,j)`；
9. 对该时间的所有开始事件执行插入；
10. 查询一次答案，代表恰好在当前事件时刻；
11. 对该时间的所有结束事件执行删除；
12. 再查询一次答案，代表当前时刻之后到下个事件之前；
13. 输出扫描过程中最大的当前最短长度。

## 正确性说明

对于任意时间 $t$，分两种情况。

### 情况一：$t$ 是事件时间 $T$

经过时间区间是闭区间。

因此：

- 在 $T$ 开始的沙虫已经活跃；
- 在 $T$ 结束的沙虫此时仍然活跃。

算法在同一时间：

1. 先加入开始事件；
2. 查询答案；
3. 再删除结束事件。

第一次查询时，`multiset` 恰好包含时刻 $T$ 所有活跃沙虫。

### 情况二：$t$ 位于相邻事件之间

设：

$$
T_i<t<T_{i+1}
$$

这段时间内没有开始或结束事件，活跃集合保持不变。

算法处理完 $T_i$ 的删除事件后再次查询。此时集合正好等于整个开区间：

$$
(T_i,T_{i+1})
$$

中的活跃集合。

所以算法覆盖了所有可能不同的活跃集合状态。

每个状态中：

```cpp
*cur.begin()
```

就是当前能召唤的沙虫长度。

对所有状态取最大值，即为全局最优答案。

## 代码实现要点

### 分数结构

```cpp
struct node1{
    ll p,q;
};
```

`p/q` 表示一个时间。

例如：

```cpp
node1 s{max(0LL,x-r),v};
```

表示：

$$
\frac{\max(0,x-r)}{v}
$$

### 交叉相乘比较

```cpp
bool ok1(const node1& a,const node1& b){
    return (i128)a.p*b.q < (i128)b.p*a.q;
}
```

比较：

$$
\frac{a.p}{a.q}<\frac{b.p}{b.q}
$$

转化为：

$$
a.p\times b.q<b.p\times a.q
$$

乘法使用 `__int128` 防止溢出。

### 判断同一时间

```cpp
bool ok2(const node1& a,const node1& b){
    return (i128)a.p*b.q == (i128)b.p*a.q;
}
```

分数不必先约分。

### 同时间事件分组

```cpp
int j = i;

while(j < m && ok2(a[i].time,a[j].time)){
    j++;
}
```

`[i,j)` 是一整段相同时间的事件。

### 段内下标

段内必须使用当前事件 `k`：

```cpp
for(int k = i;k < j;++k){
    if(a[k].op == 1){
        cur.insert(a[k].len);
    }
}
```

不能写成：

```cpp
if(a[i].op == 1)
```

`i` 是段首，`k` 才是当前事件。

### 删除一个重复元素

```cpp
auto it = cur.find(a[k].len);
cur.erase(it);
```

不能写：

```cpp
cur.erase(a[k].len);
```

后者会删除所有相同长度。

## 边界与常见坑

1. $l_i>x$ 的沙虫永远不会经过玩家；
2. 时间不要直接使用 `double`；
3. 分数乘法比较使用 `__int128`；
4. 同时间事件必须整体分组；
5. 闭区间必须先加入，再查询；
6. 结束事件删除前，该沙虫在当前时刻仍然活跃；
7. 删除后还要查询一次；
8. `multiset` 为空时不能访问 `begin()`；
9. 删除时只删除一个相同长度；
10. 段内遍历必须使用 `a[k]`；
11. 不要把段首事件类型应用到整个事件段。

## 参考代码

```cpp
#include <bits/stdc++.h>
#define code using
#define by namespace
#define Mxiaocao std
code by Mxiaocao;

using ll = long long;
using i128 = __int128;

struct node1{
    ll p,q;
};

struct node2{
    node1 time;
    int op;
    ll len;
};

bool ok1(const node1& a,const node1& b){
    return (i128)a.p*b.q < (i128)b.p*a.q;
}

bool ok2(const node1& a,const node1& b){
    return (i128)a.p*b.q == (i128)b.p*a.q;
}

void solve(){
    int n;
    ll x;
    cin >> n >> x;

    vector<node2> a;
    a.reserve(2*n);

    for(int i = 0;i < n;++i){
        ll l,r,v;
        cin >> l >> r >> v;

        if(l > x){
            continue;
        }

        ll len = r-l;

        node1 s{max(0LL,x-r),v};
        node1 t{x-l,v};

        a.push_back({s,1,len});
        a.push_back({t,-1,len});
    }

    sort(a.begin(),a.end(),[](const auto& x,const auto& y){
        return ok1(x.time,y.time);
    });

    multiset<ll> cur;
    ll res = 0;

    int m = a.size();

    for(int i = 0;i < m;){
        int j = i;

        while(j < m && ok2(a[i].time,a[j].time)){
            j++;
        }

        for(int k = i;k < j;++k){
            if(a[k].op == 1){
                cur.insert(a[k].len);
            }
        }

        if(!cur.empty()){
            res = max(res,*cur.begin());
        }

        for(int k = i;k < j;++k){
            if(a[k].op == -1){
                auto it = cur.find(a[k].len);
                cur.erase(it);
            }
        }

        if(!cur.empty()){
            res = max(res,*cur.begin());
        }

        i = j;
    }

    cout << res << '\n';
}

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    solve();

    return 0;
}
```
