## 题意简述

给定 $n$ 个带编号的节点，第 $i$ 个节点带有一个非负整数：

$$
a_i
$$

需要将这些节点连接成一棵有根树。

设节点 $i$ 到根节点经过的边数，也就是节点深度为：

$$
d_i
$$

节点 $i$ 最终产生：

$$
a_i+d_i
$$

要求最大化所有节点产生的数所构成集合的 MEX，并输出最大 MEX 和每个节点的父亲编号。

父亲数组中：

- $f_i=0$ 表示节点 $i$ 是根；
- $f_i=x$ 表示节点 $i$ 的父亲是节点 $x$。

题目要求输出的是父亲编号，而不是节点深度。

## 第一反应

理解题意后，我想到先按照 $a_i$ 排序，让排序后的第 $i$ 个节点尝试制造数字 $i$。

由：

$$
a_i+d_i=i
$$

推出：

$$
d_i=i-a_i
$$

因此写出了：

```cpp
int cur = i-a[i].val;
```

这个关系是正确的，但我最初把目标深度误当成父亲节点编号，直接将 cur 放入最终答案，导致构造错误。

## 暴力瓶颈

直接枚举树或所有节点深度均不可行。

题目结果只依赖节点深度，所以正确顺序是：

```text
先确定关键节点的目标深度
→ 再把目标深度转换成父亲关系
```

## 核心转化

要使 MEX 至少为 $m$，必须制造：

$$
0,1,2,\dots,m-1
$$

排序后让第 $i$ 个节点制造数字 $i$：

$$
a_i+d_i=i
$$

所以：

$$
d_i=i-a_i
$$

## 扫描条件

制造数字 $0$ 必须满足：

$$
a_0=0,\qquad d_0=0
$$

该节点作为根。

对于 $i>0$，节点不能再成为根，所以：

$$
d_i\ge1
$$

即：

$$
a_i<i
$$

因此：

- $i=0$ 要求 $a_0=0$；
- $i>0$ 要求 $a_i<i$。

连续满足条件的前缀长度就是候选最大 MEX。

## 为什么扫描结果是上界

如果 MEX 至少为 $m$，必须存在不同节点制造：

$$
0,1,\dots,m-1
$$

制造 $0$ 的节点是值为 $0$ 的根。

制造任意正数 $x$ 的节点不是根，所以：

$$
d\ge1
$$

由：

$$
a+d=x
$$

得到：

$$
a<x
$$

因此，若要制造 $0,1,\dots,i$，排序后的第 $i$ 个值必须满足：

$$
a_i<i
$$

一旦扫描失败，就不可能获得更大的 MEX。

## 从目标深度构造父亲

维护：

```cpp
nodeAtDepth[d]
```

表示一个已经构造出的、深度为 $d$ 的节点编号。

当前节点需要深度 depth 时：

```cpp
parent[a[i].id] = nodeAtDepth[depth-1];
nodeAtDepth[depth] = a[i].id;
```

nodeAtDepth[d] 保存的是节点编号，不是深度。

## 为什么上一层一定存在

定义：

$$
d_i=i-a_i
$$

因为排序后：

$$
a_i\ge a_{i-1}
$$

所以：

$$
d_i-d_{i-1}=1-(a_i-a_{i-1})\le1
$$

目标深度最多增加一层。

深度从 $0$ 开始，因此每次需要深度 $d_i>0$ 时，深度 $d_i-1$ 一定在此前出现过。

## 处理剩余节点

前 mex 个节点负责制造：

$$
0,1,\dots,mex-1
$$

其余节点全部挂到根。

当 mex>0 时，剩余节点满足：

$$
a_i\ge mex
$$

挂到根后产生：

$$
a_i+1\ge mex+1
$$

不会补出缺失的 mex。

当 mex=0 时，不存在值为 $0$ 的节点，任选一个节点作为根，其余节点挂根，仍然不会产生 $0$。

## 正确性证明

设扫描结果为 $m$。

对于 $0\le i<m$，令第 $i$ 个排序节点深度为：

$$
d_i=i-a_i
$$

它产生：

$$
a_i+d_i=i
$$

所以集合包含：

$$
0,1,\dots,m-1
$$

剩余节点不会产生 $m$，因此 MEX 为 $m$。

如果存在 MEX 至少为 $m+1$ 的构造，则排序后的第 $m$ 个值必须满足：

$$
a_m<m
$$

但扫描在位置 $m$ 失败，矛盾。

所以 $m$ 是最大 MEX。

## 正确做法步骤

1. 保存值和原编号；
2. 按照值排序；
3. 扫描最长合法前缀；
4. 合法前缀长度为最大 MEX；
5. 选择第一个节点作为根；
6. 计算关键节点目标深度；
7. 使用上一层代表节点作为父亲；
8. 更新当前层代表；
9. 将剩余节点全部挂根；
10. 按原编号输出父亲数组。

## 代码实现要点

- 排序时保留原编号；
- nodeAtDepth[d] 保存深度为 $d$ 的节点编号；
- parent[id] 才是最终输出；
- mex=0 时也只能有一个根；
- 不能遗漏剩余节点；
- 最终按原编号输出。

## 提交通过代码

以下代码是在理解提示后独立完成并提交通过的实现。

```cpp
#include <bits/stdc++.h>
#define code using
#define by namespace
#define Mxiaocao std
code by Mxiaocao;

using ll = long long;

struct node{
    int val,id;

    bool operator<(const node& o) const {
        if(val == o.val){
            return id < o.id;
        }
        return val < o.val;
    }
};

void solve(){
    int n;
    cin >> n;

    vector<node> a(n);

    for(int i = 0;i < n;++i){
        cin >> a[i].val;
        a[i].id = i+1;
    }

    sort(a.begin(),a.end());

    int mex = 0;

    for(int i = 0;i < n;++i){
        if(i == 0){
            if(a[i].val != 0){
                break;
            }
        }else{
            if(a[i].val >= i){
                break;
            }
        }

        mex++;
    }

    vector<int> par(n+1);
    vector<int> nodeAtDepth(n+1,-1);

    int root = a[0].id;
    par[root] = 0;
    nodeAtDepth[0] = root;

    if(mex == 0){
        for(int i = 1;i < n;++i){
            par[a[i].id] = root;
        }
    }else{
        for(int i = 1;i < mex;++i){
            int depth = i-a[i].val;

            par[a[i].id] = nodeAtDepth[depth-1];
            nodeAtDepth[depth] = a[i].id;
        }

        for(int i = mex;i < n;++i){
            par[a[i].id] = root;
        }
    }

    cout << mex << '\n';

    for(int i = 1;i <= n;++i){
        cout << par[i] << " \n"[i == n];
    }
}

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;

    while(t--){
        solve();
    }

    return 0;
}
```
