## 题意简述

需要构造一个 $n\times n$ 的矩阵，满足：

1. 矩阵中恰好使用一次：

$$
1,2,\dots,n^2
$$

2. 对于每个位置 $(i,j)$：

$$
A_{i,j}\ne i\times j
$$

当 $n=1$ 时无解，输出 `-1`。

## 第一反应

最开始想到过一种构造，但感觉不可靠，因此没有继续使用，也没有补造具体内容。

之后改为从小矩阵逐层扩展。

先构造合法的 $2\times2$ 矩阵：

```text
2 3
4 1
```

再从 $(j-1)\times(j-1)$ 扩展到 $j\times j$，保留左上角旧矩阵，只填写新增的第 $j$ 列和第 $j$ 行。

## 逐层扩展构造

扩展到 $j\times j$ 时，新数字从：

$$
(j-1)^2+1
$$

开始。

先沿第 $j$ 列从下往上填写，再沿第 $j$ 行从右往左填写。

新增格子数量为：

$$
j^2-(j-1)^2=2j-1
$$

使用数字区间为：

$$
(j-1)^2+1,\dots,j^2
$$

所以数字不会重复或遗漏。

## 第一次 WA

扩展到 $3\times3$ 时得到：

```text
2 3 7
4 1 6
9 8 5
```

其中：

$$
g_{2,3}=6=2\times3
$$

导致 WA1。

交换：

```cpp
swap(g[1][3],g[2][3]);
```

得到：

```text
2 3 6
4 1 7
9 8 5
```

两个位置都合法，修改后提交 AC。

## 新增列的冲突分析

位置 $(i,j)$ 填入：

$$
A_{i,j}=j^2-j-i+2
$$

若发生冲突：

$$
j^2-j-i+2=ij
$$

则：

$$
i=j-2+\frac4{j+1}
$$

因为 $j\ge3$，只有：

$$
j+1=4
$$

即：

$$
j=3,\qquad i=2
$$

所以新增列唯一可能冲突的位置是：

$$
(2,3)
$$

## 新增行的冲突分析

位置 $(j,i)$ 填入：

$$
A_{j,i}=j^2+1-i
$$

若发生冲突：

$$
j^2+1-i=ji
$$

则：

$$
i=j-1+\frac2{j+1}
$$

当 $j\ge3$ 时，$j+1$ 不可能整除 $2$。

所以新增行永远不会发生冲突。

旧矩阵始终保持不变，因此一次交换就能修复全部冲突。

## AC 后补充的循环左移构造

AC 后，教练补充了更简单的统一构造：

```text
1,2,3,...,n^2
```

循环左移一位：

```text
2,3,4,...,n^2,1
```

再按照行优先顺序填入矩阵。

需要明确：

```text
该构造不是用户赛时想到的方案，
而是 AC 后复盘时补充的更简洁方案。
```

除最后位置外：

$$
A_{i,j}=(i-1)n+j+1
$$

并且：

$$
A_{i,j}-ij=(i-1)(n-j)+1\ge1
$$

所以：

$$
A_{i,j}>ij
$$

最后位置：

$$
A_{n,n}=1\ne n^2
$$

因此当 $n\ge2$ 时整个构造合法。

## 两种构造比较

逐层扩展构造是用户自己完成并 AC 的方案，能够体现从小规模合法解逐层扩展的思路，但需要特殊位置修复，证明较复杂。

循环左移构造形式统一、不需要特判，可以边计算边输出，证明更简洁，是 AC 后补充的优化方案。

$$

$$

这说明构造题除了检查全局资源，还必须检查每个位置的局部限制。

## 正确做法步骤

逐层扩展方案：

1. $n=1$ 时输出 `-1`；
2. 初始化合法的 $2\times2$；
3. 逐层填写新增列和新增行；
4. 当 $n\ge3$ 时交换 $(1,3)$ 和 $(2,3)$；
5. 输出矩阵。

循环左移方案：

1. $n=1$ 时输出 `-1`；
2. 按行输出 $2,3,\dots,n^2,1$；
3. 使用差值公式统一证明。

## 自己 AC 的代码

```cpp
#include <bits/stdc++.h>
#define code using
#define by namespace
#define Mxiaocao std
code by Mxiaocao;

using ll = long long;

void solve(){
    int n;
    cin >> n;

    if(n == 1){
        cout << -1 << '\n';
        return;
    }

    vector<vector<int>> g(n+1,vector<int>(n+1));

    g[1][1] = 2;
    g[1][2] = 3;
    g[2][1] = 4;
    g[2][2] = 1;

    for(int j = 3;j <= n;++j){
        int cur = (j-1)*(j-1)+1;

        for(int i = j;i >= 1;--i){
            g[i][j] = cur;
            cur++;
        }

        for(int i = j-1;i >= 1;--i){
            g[j][i] = cur;
            cur++;
        }
    }

    if(n >= 3){
        swap(g[1][3],g[2][3]);
    }

    for(int i = 1;i <= n;++i){
        for(int j = 1;j <= n;++j){
            if(j > 1){
                cout << ' ';
            }

            cout << g[i][j];
        }

        cout << '\n';
    }
}

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    solve();

    return 0;
}
```

## 循环左移参考实现

以下实现是 AC 后补充方案，不是用户赛时代码：

```cpp
#include <bits/stdc++.h>
#define code using
#define by namespace
#define Mxiaocao std
code by Mxiaocao;

using ll = long long;

void solve(){
    int n;
    cin >> n;

    if(n == 1){
        cout << -1 << '\n';
        return;
    }

    int cur = 2;

    for(int i = 1;i <= n;++i){
        for(int j = 1;j <= n;++j){
            if(j > 1){
                cout << ' ';
            }

            if(i == n && j == n){
                cout << 1;
            }else{
                cout << cur;
                cur++;
            }
        }

        cout << '\n';
    }
}

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    solve();

    return 0;
}
```
