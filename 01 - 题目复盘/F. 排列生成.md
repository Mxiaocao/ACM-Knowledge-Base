# F. 排列生成

## 题目

[题目链接](https://ac.nowcoder.com/acm/contest/133876/F)

给定一个 $0$ 到 $n-1$ 的排列 $P$，每次操作可以选择一个整数 $d$，将所有元素统一加上 $d$ 后对 $n$ 取模（即 $P_i \leftarrow (P_i + d) \bmod n$）。要求通过一次操作使某个指定位置 $k$ 的值变为 $x$，且操作后 $P$ 仍是一个排列。

## 核心思路

**模 $n$ 整体循环平移。**

对排列 $P$ 的每个元素统一加上常数 $d$（模 $n$）：

$$
P'_i = (P_i + d) \bmod n
$$

这个操作有两个关键性质：

1. **保持排列**：循环平移是 $\{0, 1, \dots, n-1\}$ 上的双射，平移后所有元素仍互异且值域不变；
2. **差值模 $n$ 不变**：任意两个元素的差值模 $n$ 的同余类保持不变。

因此只需令 $d \equiv x - P_k \pmod n$，取 $d = (x - P_k) \bmod n$（注意负数取模），即可满足要求。

## 推导

要使位置 $k$ 的值变为 $x$：

$$
P'_k \equiv x \pmod n
$$

即：

$$
P_k + d \equiv x \pmod n
$$

因此：

$$
d \equiv x - P_k \pmod n
$$

取 $d = (x - P_k) \bmod n$，保证 $d \in [0, n-1]$。

C++ 中注意负数取模的处理：

```cpp
int d = (x - P[k]) % n;
if (d < 0) d += n;
```

或使用：

```cpp
int d = ((x - P[k]) % n + n) % n;
```

## 证明

### 1. 循环平移保持排列

映射 $f: a \mapsto (a + d) \bmod n$ 是 $\mathbb{Z}_n$ 上的平移变换，是一个双射。原排列 $P$ 的所有元素互异且值域为 $\{0, 1, \dots, n-1\}$，平移后：

- 互异性：若 $P'_i = P'_j$，则 $(P_i + d) \bmod n = (P_j + d) \bmod n$，即 $P_i \equiv P_j \pmod n$，由于 $P_i, P_j \in [0, n-1]$，必有 $P_i = P_j$，与排列互异性矛盾；
- 值域：每个元素模 $n$ 后仍在 $[0, n-1]$ 中，且互异性保证恰好覆盖全部 $n$ 个值。

因此 $P'$ 仍是 $0$ 到 $n-1$ 的排列。

### 2. 每一对差值只改变 $n$ 的倍数

设 $P'_i = P_i + d - q_i n$，其中 $q_i \in \{0, 1\}$ 为取模时的回绕修正（当 $P_i + d \geq n$ 时 $q_i = 1$，否则 $q_i = 0$）。则：

$$
\begin{aligned}
P'_j - P'_i &= (P_j + d - q_j n) - (P_i + d - q_i n) \\
&= P_j - P_i + (q_i - q_j)n
\end{aligned}
$$

因此：

$$
P'_j - P'_i \equiv P_j - P_i \pmod n
$$

差值模 $n$ 的同余类在平移下保持不变。

### 3. 整个 $f(P)$ 同余

对于任意仅依赖差值模 $n$ 的衡量函数 $f(P)$，有 $f(P') \equiv f(P) \pmod n$。这保证了平移操作不会改变排列的"差值结构"在模意义下的性质。

## 算法

1. 读入 $n$、排列 $P$、目标位置 $k$、目标值 $x$；
2. 计算 $d = (x - P_k) \bmod n$（注意负数取模）；
3. 对所有 $i \in [0, n-1]$：$P'_i = (P_i + d) \bmod n$；
4. 输出 $P'$。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k, x;
    cin >> n >> k >> x;
    vector<int> P(n);
    for (int i = 0; i < n; i++) {
        cin >> P[i];
    }

    // 计算平移量（处理负数取模）
    int d = ((x - P[k]) % n + n) % n;

    // 整体循环平移
    for (int i = 0; i < n; i++) {
        P[i] = (P[i] + d) % n;
    }

    for (int i = 0; i < n; i++) {
        cout << P[i] << " \n"[i == n - 1];
    }

    return 0;
}
```

## 复杂度

- 时间：$O(n)$，遍历一次数组。
- 空间：$O(n)$，存储排列。

## 总结

本题核心是**模循环平移**构造：

- 不要关注局部修改，寻找**整体变换**（统一加常数）；
- 利用**模意义下的不变量**（差值同余）判断操作合法性；
- 平移量 $d$ 直接从目标条件 $P_k + d \equiv x \pmod n$ 解出。
