# 分数比较

## 这个节点记录什么

记录在算法题中需要精确比较有理数（分数）时，不使用浮点数而是用交叉相乘的方法。

## 为什么不用 double

浮点数存在精度误差，在需要判断相等、排序或严格比较时容易出错，尤其当分子分母较大时。

## 方法：交叉相乘

比较两个分数 $\frac{a}{b}$ 和 $\frac{c}{d}$（$b,d>0$）：

**小于比较：**

$$
\frac{a}{b}<\frac{c}{d}
\iff a\times d < c\times b
$$

**等于比较：**

$$
\frac{a}{b}=\frac{c}{d}
\iff a\times d = c\times b
$$

## 防溢出

分子分母可能达到 $10^9$ 级别时，乘积 $a\times d$ 可能达到 $10^{18}$ 级别，超出 `long long` 范围。

需要使用 `__int128` 进行中间乘法：

```cpp
using i128 = __int128;

bool less(const node& a, const node& b) {
    return (i128)a.p * b.q < (i128)b.p * a.q;
}

bool equal(const node& a, const node& b) {
    return (i128)a.p * b.q == (i128)b.p * a.q;
}
```

## 不需要约分

交叉相乘比较时，分数不需要先约分。因为 $\frac{a}{b}$ 和 $\frac{ka}{kb}$ 在交叉相乘时结果等价。
