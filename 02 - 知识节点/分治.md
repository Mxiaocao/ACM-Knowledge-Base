# 分治

## 它是什么

分治是将大问题拆分成多个规模更小的同类子问题，分别求解后再合并答案的方法。

分治的基本结构：

```text
拆分大问题
→ 递归解决多个子问题
→ 合并子问题答案
```

**关键区分**：分治的多个递归调用不是可选分支，而是每个都必须执行。

## 常见题目特征

- 问题可以分成两个或多个独立子问题。
- 子问题与原问题结构相同，规模更小。
- 子问题的答案可以被合并为原问题的答案。
- 常见于区间问题、排序问题和几何问题。

## 基础示例

区间和：

```cpp
long long getSum(int l,int r){
    if(l == r){
        return a[l];
    }

    int mid = (l + r) / 2;

    long long leftSum = getSum(l,mid);
    long long rightSum = getSum(mid + 1,r);

    return leftSum + rightSum;
}
```

左右两个递归调用不是两个可选分支，而是左右两部分都必须处理。

区间最大值：

```cpp
int getMax(int l,int r){
    if(l == r){
        return a[l];
    }

    int mid = (l + r) / 2;

    int leftMax = getMax(l,mid);
    int rightMax = getMax(mid + 1,r);

    return max(leftMax,rightMax);
}
```

## 与回溯的区别

- 回溯：尝试多个选择分支，找到一个解或全部解，分支之间是"或"的关系。
- 分治：所有子问题都必须求解，子答案之间是"和"的关系。

## 归并排序

归并排序是分治的经典应用，详见 [[归并排序]]。

## 常见应用

- 区间求和、求最大值；
- 归并排序；
- 快速幂；
- 最近点对；
- 平面分治。
