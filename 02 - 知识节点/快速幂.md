# 快速幂

## 它是什么

用 $O(\log b)$ 时间计算 $a^b \bmod M$ 的算法。

## 核心实现

```cpp
ll qpow(ll a, ll b) {
    ll res = 1;
    while (b) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return res;
}
```

## 常见用途

- 计算模逆元（费马小定理）：$a^{M-2}$。
- 计算组合数、期望分数等。
