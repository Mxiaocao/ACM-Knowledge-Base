# 按深度维护代表节点

## 核心思想

维护：

```cpp
nodeAtDepth[d]
```

表示一个已经构造出的、深度为 $d$ 的节点编号。

当新节点需要深度 $d$ 时：

```cpp
parent[x] = nodeAtDepth[d-1];
nodeAtDepth[d] = x;
```

## 注意事项

- 保存的是节点编号，不是深度；
- 必须证明深度 $d-1$ 已经存在；
- 构造完成后仍需检查所有节点是否进入同一棵树。
