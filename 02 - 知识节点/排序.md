# 排序

## 是什么

排序是按照指定规则重新排列元素，使数据具有单调或结构化顺序的基础操作。

常见形式：

```cpp
sort(a.begin(),a.end());
```

默认按照从小到大排序。

## 解决什么问题

排序经常用于：

* 将数轴上的点按位置排列；
* 让相邻关系具有明确含义；
* 配合双指针、贪心、二分和差分；
* 将无序集合转化为序列结构。

## 常见思路

面对与距离、区间、相邻关系有关的问题时，可以先考虑：

```text
无序元素
→ 排序
→ 只分析相邻元素或连续区间
```
