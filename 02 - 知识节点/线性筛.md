# 线性筛

## 它是什么

用 $O(n)$ 时间预处理 $1$ 到 $n$ 中每个数的最小质因子 $spf[i]$ 的算法。

## 常见用途

- 快速质因数分解。
- 递推数论函数（如 $\Omega(n)$、$P(n)$、$\pi(n)$ 等）。

## 核心实现

```cpp
for (int i = 2; i <= n; ++i) {
    if (!spf[i]) {
        spf[i] = i;
        primes.push_back(i);
    }
    for (auto p : primes) {
        if (1LL * i * p > n) break;
        spf[i * p] = p;
        if (i % p == 0) break;
    }
}
```
