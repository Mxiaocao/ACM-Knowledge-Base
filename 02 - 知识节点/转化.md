## E. Edit Set 中的转化

本题将原本的合并过程转化为排序后链上的选边问题：

```text
合并 n 个元素直到剩余 k 个
→ 链上选择 n-k 条边
→ 选择最小的 n-k 个相邻差值
```

相关题目：[[E. Edit Set]]

## C. 0mar and Alternating Sums 中的转化

本题将子序列计数转化为按值分组奇偶选择：

```text
选择一个子序列
→ 每种数值选择奇数个或偶数个
→ 不同数值之间用乘法原理组合
```

相关题目：[[C. 0mar and Alternating Sums]]

## C. Mex Tree 中的转化

本题将树边选择转化为深度设计：

```text
选择合适的树边
→ 先设计目标深度
→ 再由深度映射到父亲
```

相关题目：[[C. Mex Tree]]
