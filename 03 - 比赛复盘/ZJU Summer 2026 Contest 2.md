## 七、提交与 Debug 复盘

本场只有 B 进入提交阶段。

B 发生多次 WA，但准确次数已经不记得。

这类重复失败更可能说明：

- 构造规律本身错误；
- 某个必要性质没有满足；
- 只验证了样例，没有验证最小反例；
- 代码正确实现了一个错误算法。

下次构造题连续失败时应执行：

```text
写出构造必须满足的性质
→ 枚举极小规模
→ 主动构造反例
→ 验证当前模式
→ 确认模式正确后再继续实现
```

C 没有提交，因此其问题发生在算法或实现形成之前，而不是 Debug 阶段。
