# 当前训练队列

## 作用

本文件只维护近期真正准备做的少量题目。

完整的 200 题不会全部复制到这里。

建议同时激活：

- 最少 3 题；
- 最多 15—25 题；
- 优先集中在同一个主模型或子模型。

## 当前状态

当前尚未从完整题库正式激活第一批题目。

首次选题应优先从：

[[01 - Div2B 与 Div3C 基础模型题库]]

中选择。

首批建议模型：

1. M3 贪心、排序与配对；
2. M5 前缀、差分与区间压缩。

具体题目由后续训练规划任务确定，不由 Operator 自行选择。

## 使用规则

1. Operator 不自行向队列加入题目；
2. 由用户或 Architect 明确选出首批题目后再写入；
3. 做题前不提前创建单题复盘；
4. 做题后更新真实状态；
5. 普通独立 AC 题可只进入每日复盘；
6. 有代表性的独立 AC 题可以建立单题复盘；
7. 提示后完成、看题解后完成、未补懂的题进入二刷清单；
8. 回炉题在二刷状态中写为"回炉中"；
9. 交互题不与普通题混合统计；
10. C 级资料不进入当前队列。

## 流转方式

```text
完整模型题库
→ 当前训练队列
→ 实际做题
→ 每日复盘
→ 判断是否需要单题复盘
→ 判断是否进入二刷 / 回炉
→ 回填当前队列状态
```

## 状态定义

### 做题状态

- 未开始
- 已读题
- 尝试未提交
- AC
- 提示后完成
- 看题解后完成
- 未补懂
- 暂时跳过

### 二刷状态

- 不适用
- 未二刷
- 二刷中
- 已掌握
- 回炉中
