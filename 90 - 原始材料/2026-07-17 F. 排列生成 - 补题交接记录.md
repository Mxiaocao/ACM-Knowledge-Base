## 完整证明

1. **循环平移保持排列**：映射 $a \mapsto (a + d) \bmod n$ 是 $\mathbb{Z}_n$ 上的双射，保持互异性和值域。
2. **差值同余不变**：$P'_j - P'_i = P_j - P_i + (q_i - q_j)n$，因此 $P'_j - P'_i \equiv P_j - P_i \pmod n$。
3. **平移量推导**：$d \equiv x - P_k \pmod n$，取 $d = (x - P_k) \bmod n$。
4. **负数取模处理**：C++ 中 `%` 对负数结果为负，需 `((x - P[k]) % n + n) % n`。
